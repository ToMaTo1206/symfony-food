# 🍎 Gestion d'Inventaire Alimentaire

Ce projet est une application web développée avec **Symfony**. Son objectif principal est de permettre aux utilisateurs de gérer leur stock d'aliments à la maison afin de réduire le gaspillage alimentaire.

## 🛠 Stack Technique

- **Framework** : Symfony 7
- **Langage** : PHP 8.2+
- **Base de données** : MySQL / PostgreSQL / SQLite (via Doctrine)
- **Frontend** : Twig, AssetMapper (Stimulus + Turbo)
- **Qualité de code** : PHP CS Fixer

## ⚙️ Installation

Prérequis :
- PHP 8.2 ou supérieur
- Composer
- Symfony CLI (recommandé)
- Docker (optionnel, pour la base de données)

### 1. Cloner le projet
```bash
git clone [https://github.com/TON_PSEUDO/TON_REPO.git](https://github.com/TON_PSEUDO/TON_REPO.git)
cd TON_REPO
composer install
composer dump-autoload
composer db
